# OpenShift DevOps Workshop

This is an introduction workshop for DevOps on OpenShift. This is an update from [https://github.com/openshift-labs/devops-guides](https://github.com/openshift-labs/devops-guides) to reflect the changes in OpenShift 4.x. 

To setup the tools needed in OpenShift, please refer to [appdev-templates](https://github.com/jeejeejango/appdev-templates). 

The cloud-native-demo used in this workshop is available at https://github.com/jeejeejango/cloud-native-demo


To view the guide,  [start here](guide/devops-intro.md)
